# Speed Trap

Speed Trap is an application built around Home Assistant and 
a Unifi Protect NVR that uses HA's MQTT Broker as a message
queue to execute a series of tasks initiated by an HA Automation:

## Tasks
* Create a working folder
* Export Home Assistant sensor logs
* Export video clips from Unifi Protect
* Analyze videos using OpenCV
* Generate a .NET Razor Page partial
* Publish results
